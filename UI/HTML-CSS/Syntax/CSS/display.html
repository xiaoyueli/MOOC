display 1. display: block|inline|inline-block|none|flex（详见flex标签） block: 1. 段前段后换行显示 2. set width and height, yes 3. 默认父元素宽度 default block tags: div, p, h1-h6, ul, form... inline: 1. 默认内容宽度 2. 不可设置宽高 3. 同行显示, 行内换行 default inline tags: span, a, label, cite, em.. inline-block: 1. 默认内容宽度 2. 可设置宽高 3. 同行显示，整块换行 default inline-block tags: input, textarea, select, button display: none 不显示此元素，且不占据位子 visibility：hidden 仍占据位子，只是不显示
